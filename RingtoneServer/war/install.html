Ringtones installation instructions


How to Install .Apk Apps from website.
1. Make sure you have allowed Unknown Sources.
2. Go to http://ringdroid.me/app.html and download the .Apk
3. Select the .apk from your download history
4. If you get a pop up notifying you of a blocked install, follow the steps below.
5. Click the install button on the bottom.
6. Open app, your done.

Install Blocked
1. Open settings
2. Check the Unknown Sources box
3. Go back and try again.

Allow Unknown Sources

Before you start installing application you need to allow the installation of non-Market applications.

1. Open the list of applications.
2. Open your Settings.
3. You will see a list of different settings for your phone (wireless, call settings, sound & display) tap on the Applications.
4. Select the check box next to unknown sources.
5. You are ready to install non-Market apps.

How to Install .Apk Apps on your SD card

Follow these steps if you downloaded the apk to your computer then transferred it to your SD card/phone using a USB cable.

1. Install and open the Apps Installer application from the Android Market.
2. This opens a list of .apk application names in the sdcard root directory.
3. Touch the app name to initiate installation of the app.
4. The app is now installed.
